install tornado:-
pip install tornado

Run Api:-
In terminal type
python lrf.py

api will be up and runnig on http://localhost:3006

api requests shoud be as follows:-
LRF:
http://{host}:{port}/on
ex:- http://localhost:3006/lro

LRF continous data:
http://{host}:{port}/off
ex:- http://localhost:3006/lrc
